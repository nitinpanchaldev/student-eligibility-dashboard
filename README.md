# 🎓 Student Eligibility Dashboard

A modern and responsive Student Eligibility Dashboard built using HTML, CSS, and JavaScript.

This application helps evaluate student eligibility for different academic and extracurricular activities based on predefined rules such as CGPA, attendance, and department requirements.

---

## 🚀 Features

### Student Eligibility Assessment
- Evaluate students based on:
  - CGPA
  - Attendance
  - Department/Branch
  - Activity Selection

### Dynamic Rules Engine
Supports multiple activity pathways:

- Scholarship Program
- Sports Athletics
- Coding Club
- Inter-College Hackathon

Each activity has its own eligibility criteria.

### Smart Analytics Dashboard
- Total Checks
- Eligible Students
- Rejected Students
- Average Score

### Interactive Result System
- Eligibility Status
- Score Calculation
- Performance Badge System
  - Gold
  - Silver
  - Bronze
  - Needs Improvement

### Student History Management
- Add Records
- Edit Records
- Delete Records
- Search Records
- Pagination Support

### Data Visualization
- Chart.js Doughnut Analytics
- Eligible vs Rejected Distribution

### Export Features
- CSV Report Export

### User Experience Features
- Dark Mode / Light Mode
- Responsive Design
- Toast Notifications
- Dynamic Rules Panel

### Local Storage Persistence
All records are stored in the browser using LocalStorage, allowing data to remain available after page refresh.

## ⚙️ Eligibility Logic

### Scholarship Program
- CGPA ≥ 8.5
- Attendance ≥ 75%

### Sports Athletics
- CGPA ≥ 6.0
- Attendance ≥ 65%

### Coding Club
- CGPA ≥ 7.0
- Attendance ≥ 70%
- Branch must be:
  - CSE
  - IT

### Hackathon
- CGPA ≥ 6.5
- Attendance ≥ 75%
